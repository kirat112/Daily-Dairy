Full Stack Development - 1 Month Training Diary

Week 1: Frontend Basics and Styling

Day 1 - HTML Basics

Learned the fundamentals of HTML. Created basic page structures with elements like headings, paragraphs, lists, links, and images.

Day 2 - Advanced HTML and Semantic Tags

Focused on semantic HTML5 tags (header, footer, article, section, etc.). Built a sample webpage using semantic tags to enhance accessibility.

Day 3 - Introduction to CSS

Explored CSS basics: selectors, properties, and values. Practiced styling text, colors, borders, margins, and padding.

Day 4 - CSS Box Model and Layout Techniques

Studied the box model, inline vs. block elements. Used float, display, and position properties to create layouts.

Day 5 - Introduction to Tailwind CSS

Learned the utility-first approach of Tailwind CSS. Practiced using Tailwind classes to style a responsive webpage.

Day 6 - Responsive Design with Tailwind CSS

Implemented responsive design using Tailwind's built-in classes. Used breakpoints to make layouts adaptable to different screen sizes.

Day 7 - Bootstrap Basics

Started with Bootstrap components and grid system. Created a responsive webpage using Bootstrap's pre-built components.

Week 2: Frontend Framework and JavaScript

Day 8 - Advanced Bootstrap Features

Worked with Bootstrap forms, buttons, and modals. Created interactive UI elements using Bootstrap utilities.

Day 9 - JavaScript Fundamentals

Explored JavaScript syntax, variables, data types, and operators. Practiced with loops, conditional statements, and functions.

Day 10 - DOM Manipulation with JavaScript

Learned how to manipulate the DOM using JavaScript. Implemented simple dynamic interactions on a webpage.

Day 11 - ES6 Features

Introduced ES6 features like arrow functions, template literals, and destructuring. Practiced using ES6 syntax in JavaScript code.

Day 12 - Introduction to React

Set up a React environment and learned about JSX. Created functional components and rendered them to the UI.

Day 13 - React State and Props

Worked with state and props in React components. Built a small project to demonstrate state management.

Day 14 - React Lifecycle Methods and Hooks

Learned about React lifecycle methods and useState, useEffect hooks. Implemented hooks in functional components to manage state and side effects.

Week 3: Backend Basics with Node.js and Express.js

Day 15 - Introduction to Node.js

Installed Node.js and understood its runtime environment. Created a simple Node.js server using core modules.

Day 16 - NPM and Express.js Basics

Learned how to use NPM to manage packages. Set up a basic Express.js server and created RESTful routes.

Day 17 - Middleware in Express.js

Explored the concept of middleware in Express.js. Implemented middleware for logging, authentication, and error handling.

Day 18 - Building APIs with Express.js

Created a RESTful API with CRUD operations. Tested API endpoints using Postman.

Day 19 - Introduction to MongoDB

Installed MongoDB and learned about NoSQL databases. Worked with MongoDB commands to create, read, update, and delete data.

Day 20 - Connecting MongoDB with Express.js

Integrated MongoDB with Express.js using Mongoose. Created models and schemas to interact with the database.

Day 21 - Full Stack Project Setup: Finance Tracker

Set up the structure for the Finance Tracker project using React, Express.js, and MongoDB. Created the initial backend setup and connected it to the frontend.

Week 4: Full Stack Development and Project Building

Day 22 - User Authentication for Finance Tracker

Implemented user authentication with JWT (JSON Web Tokens). Created login and registration endpoints on the backend.

Day 23 - Integrating Finance Tracker Frontend and Backend

Connected the React frontend with the Express.js backend API. Implemented authentication on the frontend using JWT.

Day 24 - Finance Tracker: Expense Management Module

Developed a module to track income and expenses. Implemented CRUD operations to manage financial records in MongoDB.

Day 25 - Data Visualization with Chart.js

Added data visualization to the Finance Tracker using Chart.js. Displayed income and expenses through dynamic charts and graphs.

Day 26 - Error Handling and Data Validation

Implemented server-side validation using Express.js middleware. Added error handling mechanisms for frontend and backend.

Day 27 - Deployment Preparation

Optimized the Finance Tracker application for production. Prepared the codebase for deployment.

Day 28 - Deploying Finance Tracker to Heroku and MongoDB Atlas

Deployed the full-stack Finance Tracker application to Heroku. Configured MongoDB Atlas for the production database.

Day 29 - Final Testing and Bug Fixes

Conducted end-to-end testing of the Finance Tracker. Fixed bugs and improved performance based on user feedback.

Day 30 - Project Review and Wrap-Up

Reviewed the entire project, summarizing the key learning outcomes. Prepared project documentation and presented the Finance Tracker.
